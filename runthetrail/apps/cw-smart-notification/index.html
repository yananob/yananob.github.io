CW Smart Notification

Chatwork の未読メッセージをスマートに通知

Chatwork の未読メッセージを定期的にチェックし、自分宛てのメッセージなどをスマートに通知する Android アプリです。

主な機能

定期ポーリング

15分ごとに Chatwork API を確認し、新しい未読メッセージがあれば通知します。

自分宛てフィルタ

自分への返信や TO 指定されたメッセージのみを通知するフィルタ機能。

消灯時間設定

夜間など、通知を受け取りたくない時間帯を指定できます。

通知履歴

アプリ内で通知されたメッセージの履歴を確認し、直接 Chatwork の該当メッセージへ遷移できます。

導入方法

1. Chatwork API トークンの取得

本アプリを利用するには、Chatwork の API トークンが必要です。

  1. Chatwork にログインします。
  2. 画面右上のプロフィールアイコンをクリックし、「API 設定」を選択します。
  3. パスワードを入力して「表示」をクリックし、発行された API トークンをコピーします。

2. アプリの設定

  1. アプリを起動します。
  2. 初回起動時に通知権限の許可を求められるので、許可してください。
  3. 画面右上の設定アイコン（歯車）をタップします。
  4. 「Chatwork API トークン」欄に、先ほどコピーしたトークンを貼り付けます。
  5. 必要に応じて「自分宛てのみ」や「消灯時間を有効にする」を設定します。
  6. 「保存」ボタンをタップして保存します。
