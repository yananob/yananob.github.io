CW Smart Notification

Smart notifications for Chatwork on Android.

CW Smart Notification is an Android app that periodically checks for unread messages on Chatwork and provides smart notifications for messages directed to you.

Key Features

Periodic Polling

Checks the Chatwork API every 15 minutes and notifies you if there are new unread messages.

Mention Filter

Notifies you only of replies to you or messages where you are specified in the "TO" field.

Quiet Hours

Allows you to specify time periods, such as nighttime, when you do not want to receive notifications.

Notification History

View the history of notified messages within the app and navigate directly to the corresponding message in Chatwork.

How to Use

1. Obtain a Chatwork API Token

To use this app, you need a Chatwork API token.

  1. Log in to Chatwork.
  2. Click the profile icon in the upper right corner and select "API Settings."
  3. Enter your password, click "Display," and copy the issued API token.

2. App Configuration

  1. Launch the app.
  2. On the first launch, grant the necessary notification permissions.
  3. Tap the settings icon (gear) in the upper right corner.
  4. Paste the token you just copied into the "Chatwork API Token" field.
  5. Configure options like "Mentions Only" or "Enable Quiet Hours" as needed.
  6. Tap the "Save" button.
