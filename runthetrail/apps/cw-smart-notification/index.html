CW Smart Notification

Smart notifications for Chatwork unread messages on Android.

An Android application that periodically checks your Chatwork unread messages and smartly notifies you of mentions and relevant updates.

Key Features

Periodic Polling

Checks the Chatwork API every 15 minutes and notifies you if there are new unread messages.

Mention Filter

Filter notifications to only receive alerts for direct mentions (TO) or replies.

Quiet Hours

Specify time ranges (e.g., at night) when you don't want to be disturbed by notifications.

Notification History

Review your notification history within the app and jump directly to the message in Chatwork via deep links.

How to Setup

1. Obtain Chatwork API Token

To use this app, you need a Chatwork API token.

  1. Log in to Chatwork.
  2. Click your profile icon in the top-right corner and select "API Settings".
  3. Enter your password, click "Display", and copy the generated API token.

2. Configure the App

  1. Launch the app.
  2. Grant notification permissions when prompted on first launch.
  3. Tap the settings icon (gear) in the top-right corner.
  4. Paste your copied token into the "Chatwork API Token" field.
  5. Optionally configure "Mentions Only" or "Quiet Hours".
  6. Tap "Save" to save your settings.
